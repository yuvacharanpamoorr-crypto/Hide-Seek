How RSA Works

RSA (Rivest–Shamir–Adleman) is an asymmetric cryptographic algorithm that uses a pair of keys:

  • Public key — shared openly, used to encrypt messages.
  • Private key — kept secret, used to decrypt.
  • Security relies on the difficulty of factoring large prime numbers.

Hybrid Mode

Raw RSA can only encrypt data smaller than the key. Hybrid mode generates a random AES key, encrypts the message with AES, then encrypts the AES key with RSA — enabling encryption of arbitrarily large data.

Key Sizes

  • 512-bit — Demo only, fast generation (~1s)
  • 1024-bit — Educational, decent security (~2-5s)
  • 2048-bit — Standard security (~10-30s)
