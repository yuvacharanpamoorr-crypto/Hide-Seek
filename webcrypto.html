WebCrypto API

The Web Crypto API provides native, hardware-accelerated cryptographic operations:

  • RSA-OAEP — Optimal Asymmetric Encryption Padding, resistant to padding oracle attacks.
  • AES-256-GCM — Authenticated encryption providing confidentiality + integrity.
  • SHA-256 — Secure hashing for integrity verification.

vs Educational RSA

  • Educational — Custom BigInt, great for learning. Not production-safe.
  • WebCrypto — Browser-native, hardware-accelerated, production-ready.
