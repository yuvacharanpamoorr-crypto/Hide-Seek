How it works

This tool combines RSA asymmetric encryption with LSB steganography:

  • Step 1 — Generate an RSA key pair (public + private).
  • Step 2 — Enter your message, encrypt it with the public key (hybrid RSA+AES), then embed the ciphertext into an image using Least Significant Bit encoding.
  • Step 3 — To decode, upload the stego image and provide the private key. The hidden ciphertext is extracted and decrypted.

Why RSA?

  • No shared password — only the private key holder can decrypt.
  • You can share the public key openly.
  • Hybrid RSA+AES handles messages of any length.

Tips

  • Always use PNG — JPEG compression destroys hidden bits.
  • Larger images can hold more data.
  • Save your private key — without it, decryption is impossible.
