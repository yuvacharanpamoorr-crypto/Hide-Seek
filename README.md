# Secure Steganography and Cryptography Suite

## Overview

Secure Steganography and Cryptography Suite is a cybersecurity project that combines multiple data protection techniques into a single platform. The system provides secure communication through image steganography, RSA encryption, digital watermarking, and modern browser cryptographic APIs.

The project demonstrates how information can be hidden, encrypted, authenticated, and securely transmitted using different security mechanisms.

---

## Features

### Image Steganography

* Hide secret messages inside images.
* Extract hidden messages from encoded images.
* Maintain image quality after embedding data.

### RSA Cryptography

* Generate public and private keys.
* Encrypt sensitive information.
* Decrypt encrypted messages securely.

### Digital Watermarking

* Embed ownership information into images.
* Protect digital content from unauthorized usage.
* Verify image authenticity.

### Web Crypto API

* Browser-based cryptographic operations.
* Secure key generation.
* Encryption and decryption demonstrations.
